# 基于MCP协议的多智能体协作与任务管理平台

**项目名称：** 基于MCP协议的多智能体协作与任务管理平台（HeySure AI 2.0）  
**更新时间：** 2026-05-09

## 一、项目当前进展

目前项目已完成从概念设计到核心功能闭环的实现，整体进入联调优化与展示打磨阶段。系统已形成“前端控制台 + 后端服务 + 独立执行器”的三层架构，能够支持多 AI 角色管理、独立会话、任务调度、MCP 工具调用以及工作区上下文管理。

从实现状态来看，项目不再停留在单一聊天演示，而是已经具备较完整的 AI 协同工作流能力，可以支撑“管理员调度 AI、AI 处理任务、工具辅助执行、结果回传展示”的实际运行场景。当前重点工作主要集中在三个方面：

- 进一步稳定任务调度、会话流转和状态同步
- 继续完善 MCP 工具调用与工作区操作的可视化展示
- 为比赛申报准备项目简介、技术路线、成果展示与演示材料

## 二、已完成的阶段性成果

### 0. 项目开发阶段划分

项目整体可以概括为四个阶段：

- **阶段一：架构搭建**，完成前后端分离、Socket.IO 通信、基础鉴权与数据库接入
- **阶段二：核心功能实现**，完成 AI 卡片、会话系统、角色分层、配置管理
- **阶段三：任务与工具闭环**，完成任务调度、MCP 工具调用、工作区文件上下文能力
- **阶段四：联调与展示优化**，完善状态展示、交互体验、任务可追踪性和比赛材料整理

### 1. 核心架构已打通

- 完成前端 `web`、后端 `server`、执行器 `agent` 的分层结构搭建
- 前端使用 Vue 3 + Vite 实现统一管理界面，后端使用 FastAPI 提供 API 与调度服务
- 独立 `agent` 通过 Socket.IO 接收执行指令，承担任务执行与结果回传职责
- 使用 Docker Compose 对三项服务进行统一编排，便于本地启动与现场演示
- 项目主入口和运行链路已可从配置文件直接拉起，适合比赛场景快速展示

### 2. AI 管理能力已落地

- 支持多 AI 卡片化管理，每个 AI 对应独立配置与独立状态
- 支持 AI 的新增、编辑、删除、启动、停止，配置修改后可立即生效
- 支持按角色区分不同 AI 类型，例如主脑、辅助管理员、数字成员
- 支持按 `assistant_admin` / `digital_member` 进行一级角色划分，并进一步区分 `manager` / `member`
- 卡片中可展示运行状态、当前行为、Token、最近任务、MCP 状态等关键运行信息
- 支持从卡片直接进入设置、对话和工具查看，减少多层页面跳转

### 3. 对话与上下文管理已完善

- 支持按 `ai_config_id + ai_kind` 隔离的独立会话，避免不同 AI 之间上下文混淆
- 支持聊天历史记录加载、会话切换、新建与删除
- 支持任务会话分组展示，同名任务按代际归组，便于观察每一轮执行过程
- 支持后台 run 持续执行，即使关闭弹窗或刷新页面，任务仍可继续运行
- 支持运行中状态的实时查看与终止控制，避免任务失控或长时间挂起
- 支持按角色自动映射会话类型，减少手动配置成本

### 4. 任务系统已具备实用性

- 支持立即任务、一次性定时任务和循环任务三类调度方式
- 支持任务执行记录、任务代际追踪、任务暂停、恢复、终止和删除
- 支持任务完成后的传承与监督机制，可持续追踪任务是否真正完成
- 支持任务模板复用，可直接基于已完成任务新建后续任务
- 支持任务优先级抢占与调度切换，更符合实际多任务并行场景
- 任务执行记录中可以看到任务状态、最近运行结果、生成轮次与生命周期信息

### 5. MCP 工具调用与工作区能力已接入

- 支持 MCP 工具列表查看、工具说明展示与按需调用
- 支持工作区目录读取、文件读取、文件名检索和 Git Diff 查看
- 支持按 AI 配置限制工具权限与工作目录，避免越权访问
- 支持将工具调用结果嵌入对话消息流中，形成“提问 - 调用 - 回传 - 再决策”的闭环
- 支持结构化写文件、编辑文件和按文件名读取，方便 AI 实际操作工作区
- 工具调用过程可在界面中看到执行状态，增强可解释性和可演示性

### 6. 交互与展示层已完成优化

- 前端已完成统一控制台式界面，首页可集中展示所有 AI 状态与操作入口
- AI 卡片、任务弹窗、配置弹窗、聊天窗口等模块已经拆分，职责清晰
- 支持运行状态、实时思考、MCP 状态、任务状态的可视化展示
- 支持独立对话、工具说明、目录查看等面板式操作，便于现场讲解
- 已完成多个关键交互流程的修复与收敛，整体可演示性较强
- 在展示层面已经具备“看得见 AI 如何工作”的效果，而不仅仅是聊天界面
